# Monastery360 - React + Vite

Explore Sikkim’s monasteries digitally with immersive 360° tours, cultural archives, and community features.

---

## Features

- **Virtual Tours:** Walk through monastery interiors, prayer halls, and relic rooms in 360° panoramas.  
- **Digital Archives:** Access manuscripts, murals, and historical artifacts online.  
- **Cultural Calendar:** Track festivals, rituals, and teachings.  
- **Audio Narration:** Listen to stories and chants narrated by monks.  
- **Interactive Maps:** Explore monasteries geographically.  
- **Community & Learning:** Join discussions and access educational resources.

---

## Tech Stack

- **Frontend:** React 18 + Vite  
- **Styling:** Tailwind CSS  
- **Routing:** React Router DOM  
- **Backend:** Node.js / Express  
- **Dev Tools:** ESLint, Prettier
